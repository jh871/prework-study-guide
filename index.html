Prework Study Guide

HTML

  • The head element contrains metadata about the webpage.
  • within head only title will be visible to the user, as the page title in the browser window tab.
  • The body element represents the visible content shown to the user.
  • h1, h2, h3, h4, h5 and h6 are tiers of headings, with h1 as the largest.
  • ul and ol are unordered and ordered lists, with li written for each list item.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Multiple selectors can be assigned the same properties.
  • a full stop infront of a class selector turns it into a class attribute which applies to all matching elements.
  • colours in CSS can be done using semantic refs or hex #.
  • Listing values after a property can set properties for different axis/directions.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout branch-name: switches between branches
  • git pull origin main: pulls latest changes from remote
  • cd ..: change directory to upper directory
  • git branch -d branch-name: deletes branch (-D: force delete)
  • git push remote-name: send changes to remote
  • git branch -a: view remote branches
